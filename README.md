aGov - Packaging
================

Manifests for building:

* AWS image
* Azure image
* Containers

## Roadmap

* Azure support
* Container support
* 7.x-1.x releases
* 7.x-2.x releases

## Requirements

* Packer - http://packer.io

## Usage

**Build an AWS AMI based on aGov 7.x-3.x**

```bash
$ make ami-3
```
